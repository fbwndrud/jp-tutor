# jp-tutor

일본어 학습 튜터 Claude Code 플러그인.

`/jp` 명령 하나로 어휘/문법/한자/독해/작문/번역/회화/문화를 학습하고, JLPT 개념별 마스터리를 추적합니다.

## 설치

Claude Code에서 플러그인을 추가합니다:

```bash
claude plugin add fbwndrud/jp-tutor
```

설치 후 `/jp`가 사용 가능합니다.

## 사용법

### 첫 실행

```
/jp
```

첫 실행 시 셋업이 시작됩니다:
1. 현재 레벨 자가 평가 (완전 초보 / 기초 있음 / 중급 / 고급)
2. 학습 목표 선택 (JLPT 시험 / 여행 / 비즈니스 / 일상회화 / 자유)
3. 진단 퀴즈 4문제로 레벨 측정
4. `jp-data/progress.json`에 초기 진도 데이터 생성

### 학습 세션

인자 없이 호출하면 현재 상태를 분석해서 학습을 추천합니다:

```
/jp
```

원하는 학습을 직접 지정할 수도 있습니다:

```
/jp N4 문법 퀴즈 내줘
/jp 한자 읽기 연습
/jp 편의점에서 물건 사는 상황으로 회화 연습
/jp 작문 연습하고 싶어
/jp 진도 보여줘
```

### 8가지 학습 모드

| 모드 | 예시 입력 | 설명 |
|------|----------|------|
| 어휘 퀴즈 | `단어 퀴즈`, `N3 어휘` | 4지선다, 빈칸 채우기, 유의어 구분 |
| 문법 연습 | `문법`, `て형 연습` | 패턴 설명 → 변환/선택/오류찾기 |
| 한자 학습 | `한자`, `N4 한자` | 음독/훈독, 부수, 관련 단어 |
| 독해 | `독해`, `지문 읽기` | 레벨별 지문 + 이해 질문 |
| 작문 | `작문`, `일기 써보기` | 주제 제시 → 교정 피드백 + 모범 답안 |
| 번역 | `번역 연습` | 한↔일 양방향, 직역/의역 비교 |
| 회화 시뮬레이션 | `식당 상황 연습` | 상황별 텍스트 대화 + 사후 교정 |
| 문화/관용표현 | `관용구`, `비즈니스 매너` | 문화 배경 + 관용구 퀴즈 |

### 진도 대시보드

```
/jp 진도 보여줘
```

```
📊 학습 대시보드 — 일본어 N4

📅 시작: 2026-03-07 | 총 세션: 12회

▸ 어휘 (32개 학습)
  🟦 18개  🟩 8개  🟨 4개  🟥 2개
  약점: 経済(🟥), 政治(🟥)

▸ 문법 (15개 학습)
  🟦 8개  🟩 4개  🟨 2개  🟥 1개
  약점: 受身形(🟥) — 직접/간접 수동 구분 어려움

▸ 한자 (45자 학습)
  🟦 30자  🟩 10자  🟨 3자  🟥 2자

💡 추천: 受身形 드릴 → 어휘 약점 복습
```

## 마스터리 추적

개념 단위(개별 단어, 문법 포인트, 한자)로 숙련도를 추적합니다:

| 배지 | 정답률 | 의미 |
|------|--------|------|
| 🟥 | 0-39% | 약점 — 드릴 우선 |
| 🟨 | 40-69% | 보통 — 복습 필요 |
| 🟩 | 70-89% | 양호 |
| 🟦 | 90-100% | 마스터 |
| ⬜ | 미측정 | 아직 테스트 안 함 |

진도 데이터는 `jp-data/progress.json`에 저장됩니다. 프로젝트 디렉토리에 자동 생성되므로 `.gitignore`에 `jp-data/`를 추가하는 것을 권장합니다.

## JLPT 커리큘럼

N5부터 N1까지 내장 커리큘럼이 포함되어 있습니다:

| 레벨 | 어휘 | 한자 | 목표 |
|------|------|------|------|
| N5 | ~800어 | ~100자 | 기본 인사, 자기소개 |
| N4 | ~1,500어 | ~300자 | 일상 대화 |
| N3 | ~3,750어 | ~650자 | 뉴스/신문 이해 |
| N2 | ~6,000어 | ~1,000자 | 비즈니스 대응 |
| N1 | ~10,000어 | ~2,000자 | 전문적 토론 |

## 자료 통합

학습 자료 파일을 제공하면 자동으로 분석하여 커리큘럼에 통합합니다:

```
/jp 이 교재로 공부하자 [파일 첨부]
```

PDF, Markdown, 텍스트 파일에서 어휘/문법/한자를 추출하고 학습 세션을 제안합니다.

## 플러그인 구조

```
jp-tutor/
├── metadata.json                              # 플러그인 메타데이터
├── skills/jp/
│   ├── SKILL.md                               # 메인 튜터 가이드라인
│   └── references/
│       ├── quiz-rules.md                      # 퀴즈 생성 규칙
│       ├── exercise-types.md                  # 8가지 학습 모드 상세
│       ├── grading-rubrics.md                 # 채점/피드백 기준
│       └── curricula/
│           └── japanese-jlpt.md               # JLPT N5-N1 커리큘럼
├── slash-commands/
│   └── jp.md                                  # /jp 슬래시 커맨드
└── evals/                                     # 스킬 평가 테스트
    ├── evals.json
    └── trigger-eval.json
```
