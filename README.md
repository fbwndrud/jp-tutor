# jp-tutor

일본어 학습 튜터 Claude Cowork 플러그인.

`/jp` 하나로 JLPT, 비즈니스 일본어, 교재 기반 학습까지. 그냥 일본어 관련해서 말하면 알아서 해줍니다.

## 설치

Claude Cowork → **설정** → **플러그인** → **URL로 마켓플레이스 추가**:

```
https://github.com/fbwndrud/jp-tutor.git
```

## 사용법

**그냥 말하면 됩니다.** 명령어를 외울 필요 없습니다.

```
일본어 공부하고 싶어          → 온보딩 시작
오늘 뭐 공부하지?            → 오늘의 학습 세트
단어 퀴즈 내줘              → 어휘 퀴즈
한자 좀 외우자              → 한자 플래시카드
편의점에서 물건 사는 연습      → 회화 시뮬레이션
비즈니스 메일 쓰는 법 알려줘   → 비즈니스 모드
진도 보여줘                 → 대시보드
```

빠르게 특정 모드로 가고 싶으면 `/jp:vocab`, `/jp:biz` 같은 명령어도 있습니다.

## 주요 기능

### 3가지 학습 프로필

- **JLPT 자율 학습** — N5~N1 커리큘럼 기반
- **비즈니스 일본어** — 경어, 메일, 회의, 전화, 업종별 용어
- **교재 기반** — 교재 등록하면 수업 진도에 맞춰 예습/복습

### 8가지 학습 모드 + 3개 전문 에이전트

어휘 · 문법 · 한자 · 독해 · 작문 · 번역 · 회화 · 문화

독해/작문/회화는 전문 에이전트가 멀티턴으로 깊이 있게 진행합니다.

### 에빙하우스 복습

```
정답 → 1일 → 3일 → 7일 → 14일 → 30일 → 60일
오답 → 1일로 리셋
```

복습할 시간이 되면 자동으로 알려줍니다.

### 게이미피케이션

- XP & 30단계 레벨 (초심자 → 센세이)
- 스트릭 · 마스터리 · 약점극복 배지
- 일일/주간 목표

### 마스터리 추적

| 🟦 마스터 | 🟩 숙련 | 🟨 학습중 | 🟥 보강필요 | ⬜ 미측정 |
|----------|---------|----------|-----------|---------|

### 내보내기 & 리포트

단어장 CSV/Anki 내보내기, 주간/월간 학습 리포트도 지원합니다.

## 명령어 목록

명령어를 직접 쓸 필요는 없지만, 알아두면 편합니다:

| 명령어 | 설명 |
|--------|------|
| `/jp` | 뭐든 물어보세요 |
| `/jp:start` | 처음 시작 / 프로필 설정 |
| `/jp:daily` | 오늘의 학습 |
| `/jp:review` | 복습 |
| `/jp:vocab` | 어휘 퀴즈 |
| `/jp:grammar` | 문법 연습 |
| `/jp:kanji` | 한자 학습 |
| `/jp:translate` | 번역 연습 |
| `/jp:culture` | 문화/관용표현 |
| `/jp:biz` | 비즈니스 일본어 |
| `/jp:textbook` | 교재 기반 학습 |
| `/jp:dashboard` | 대시보드 |
| `/jp:export` | CSV/Anki 내보내기 |
| `/jp:report` | 학습 리포트 |
| `/jp:update` | 진도 동기화 |
