# jp-tutor

일본어 학습 튜터 Claude Cowork 플러그인.

`/jp` 하나로 히라가나부터 JLPT N1까지. 그냥 일본어 관련해서 말하면 알아서 해줍니다.

## 설치

Claude Cowork에서:

1. 채팅 입력창 **+** 클릭
2. **플러그인** → **플러그인 추가**
3. **개인** 탭 → **+** 클릭
4. **URL로 마켓플레이스 추가** 선택
5. URL 입력:
   ```
   https://github.com/fbwndrud/jp-tutor.git
   ```

## 사용법

**그냥 말하면 됩니다.** 명령어를 외울 필요 없습니다.

```
일본어 공부하고 싶어          → 온보딩 시작
히라가나부터 배우고 싶어       → 가나 학습 (Pre-N5)
오늘 뭐 공부하지?            → 오늘의 학습 세트
단어 퀴즈 내줘              → 어휘 퀴즈
한자 좀 외우자              → 한자 플래시카드
편의점에서 물건 사는 연습      → 회화 시뮬레이션
비즈니스 메일 쓰는 법 알려줘   → 비즈니스 모드
진도 보여줘                 → 대시보드
```

빠르게 특정 모드로 가고 싶으면 `/jp:vocab`, `/jp:biz` 같은 명령어도 있습니다.

## 주요 기능

### 완전 초보 OK — Pre-N5 가나 학습

히라가나/카타카나를 모르는 완전 초보자도 시작할 수 있습니다.

- 빈도 기반 7단계 히라가나 + 4단계 카타카나 커리큘럼
- 인터랙티브 50음도 차트 (진행률/마스터리 표시)
- 혼동 문자 비교 학습 (は/ほ, ツ/シ 등)
- 가나 마스터 시 N5로 자동 승급

### 3가지 학습 프로필

- **JLPT 자율 학습** — N5~N1 커리큘럼 기반
- **비즈니스 일본어** — 경어, 메일, 회의, 전화, 업종별 용어
- **교재 기반** — 교재 등록하면 수업 진도에 맞춰 예습/복습

### 8가지 학습 모드 + 3개 전문 에이전트

어휘 · 문법 · 한자 · 독해 · 작문 · 번역 · 회화 · 문화

독해/작문/회화는 전문 에이전트가 멀티턴으로 깊이 있게 진행합니다.

### 음성 발음 지원 (TTS)

- **Cowork (원격)**: HTML TTS 플레이어로 브라우저에서 바로 듣기
  - 일본어 음성 선택 (Kyoko, Otoya, Google 日本語 등 브라우저별 제공)
  - 전체 재생 / 반복 재생 / 속도 조절 (0.3x~2.0x)
  - 키보드 단축키 (Space: 재생, 좌우: 이전/다음)
- **CLI (로컬 macOS)**: `say` 명령어로 즉시 발음 재생
- 레벨별 속도 자동 조절 (초급은 느리게, 고급은 자연 속도)
- **발음 먼저, 퀴즈 나중** — 새 개념은 TTS로 충분히 듣고 학습한 뒤 퀴즈

### Wabi-Sabi Digital 디자인

HTML 템플릿은 일본 잉크 & 종이 미학에서 영감을 받은 통일된 디자인 시스템을 사용합니다.

- 서체: Shippori Mincho (명조 디스플레이) + Noto Sans JP (본문)
- 색상: 종이 배경, 먹 텍스트, 남색 악센트, 주색(朱色) 포인트
- 라이트/다크 모드 자동 지원
- 종이 질감 텍스처, 슬라이드업 애니메이션, 장식 코너 마크

### 에빙하우스 복습

```
정답 → 1일 → 3일 → 7일 → 14일 → 30일 → 60일
오답 → 1일로 리셋
```

복습할 시간이 되면 자동으로 알려줍니다.

### 게이미피케이션

- XP & 30단계 레벨 (초심자 → 센세이)
- 스트릭 · 마스터리 · 약점극복 · 가나 배지
- 일일/주간 목표

## 명령어 목록

명령어를 직접 쓸 필요는 없지만, 알아두면 편합니다:

| 명령어 | 설명 |
|--------|------|
| `/jp` | 뭐든 물어보세요 |
| `/jp:start` | 처음 시작 / 프로필 설정 |
| `/jp:kana` | 히라가나/카타카나 학습 |
| `/jp:daily` | 오늘의 학습 |
| `/jp:review` | 복습 |
| `/jp:vocab` | 어휘 퀴즈 |
| `/jp:grammar` | 문법 연습 |
| `/jp:kanji` | 한자 학습 |
| `/jp:translate` | 번역 연습 |
| `/jp:culture` | 문화/관용표현 |
| `/jp:biz` | 비즈니스 일본어 |
| `/jp:textbook` | 교재 기반 학습 |
| `/jp:dashboard` | 대시보드 |
| `/jp:export` | CSV/Anki 내보내기 |
| `/jp:report` | 학습 리포트 |
| `/jp:update` | 진도 동기화 |
