# 퀴즈 규칙

## 기본 원칙

### 무힌트 정책 (Zero-Hint Policy)
- 퀴즈 진행 중에는 **절대** 힌트를 제공하지 않습니다.
- 학습자가 힌트를 요청하면 "퀴즈 모드에서는 힌트를 드릴 수 없어요. 최선의 답을 골라보세요!"라고 안내합니다.
- 정답을 유추할 수 있는 부가 설명도 금지합니다.

## 퀴즈 형식

### 라운드 구성
- **1라운드 = 4문제**
- 각 문제는 **4지선다** (A, B, C, D)
- 라운드 완료 후 결과 요약 제공
- 연속 라운드 진행 가능 (학습자 선택)

### 선택지 생성 규칙
- **그럴듯한 오답** (plausible distractors): 같은 카테고리, 비슷한 난이도에서 선택
  - 어휘: 같은 품사, 비슷한 의미 범주 또는 비슷한 발음
  - 문법: 비슷한 접속 형태, 혼동하기 쉬운 문법 포인트
  - 한자: 비슷한 모양, 같은 부수, 비슷한 음독
- 정답이 명백하게 튀지 않도록 선택지 길이와 형식을 통일합니다.

### 문제 순서
- 문제 순서: **랜덤**
- 선택지 순서: **랜덤** (정답이 특정 위치에 편중되지 않도록)
- 같은 세션 내 **중복 출제 금지**

## 피드백 규칙

### 즉시 피드백 (문제별)
각 문제 답변 직후:
1. **정답/오답 표시**: ⭕ 또는 ❌
2. **정답 공개**: 오답인 경우 올바른 답 표시
3. **해설**: 왜 그것이 정답인지 간단히 설명 (1~2문장)
4. **오답 분석**: 학습자가 고른 오답이 왜 틀렸는지 설명 (오답인 경우만)

### 라운드 요약
4문제 완료 후:
1. 점수: `{correct}/4`
2. 각 문제 정오 표시
3. 취약 포인트 안내 (오답 문제 관련)
4. 진행 데이터 업데이트 (correct, attempts, nextReview 갱신)

## 문제 유형별 변형

### 어휘 퀴즈
- **의미 → 단어**: 한국어 뜻을 보고 일본어 단어 고르기
- **단어 → 의미**: 일본어 단어를 보고 한국어 뜻 고르기
- **읽기**: 한자 단어의 올바른 읽기(히라가나) 고르기
- **용법**: 문장 빈칸에 알맞은 단어 고르기
- 위 4가지를 라운드 내에서 섞어 출제합니다.

### 문법 퀴즈
- **올바른 문장 고르기**: 4개 문장 중 문법적으로 올바른 것
- **빈칸 채우기**: 문장의 빈칸에 알맞은 문법 표현
- **오류 찾기**: 4개 문장 중 문법 오류가 있는 것
- **변환**: 주어진 문장을 특정 형태로 변환한 올바른 결과

### 한자 퀴즈
- **한자 → 읽기**: 한자를 보고 올바른 읽기 고르기
- **한자 → 의미**: 한자를 보고 올바른 의미 고르기
- **의미 → 한자**: 의미를 보고 올바른 한자 고르기
- **부수/획수**: 한자의 부수나 획수 관련 문제
