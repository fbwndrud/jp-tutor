# 가나 학습 전용 연습 유형

## 1. 문자 인식 (Character Recognition)

### 형식
가나 문자를 보고 로마자/한국어 발음을 맞추기

### 진행
1. 화면에 가나 1자 표시 (예: か)
2. 4지선다: ka / sa / ta / na
3. 정답 후 해당 문자가 포함된 단어 1개 표시

### 변형
- **히라가나→발음**: ひ → hi
- **카타카나→발음**: ヒ → hi
- **히라가나↔카타카나**: ひ → ヒ (대응 문자 찾기)

---

## 2. 발음→문자 (Sound to Character)

### 형식
발음(로마자)을 보고 올바른 가나 문자 고르기

### 진행
1. 발음 표시 (예: "shi")
2. 4지선다: し / す / ち / さ
3. 정답 시 관련 단어 표시

---

## 3. 단어 읽기 (Word Reading)

### 형식
가나로 쓰인 단어를 읽고 뜻 맞추기 (Stage 2부터)

### 진행
1. 가나 단어 표시 (예: さくら)
2. 4지선다: 벚꽃 / 사쿠라 / 산 / 강
3. 또는 직접 입력 (로마자/한국어)

### 난이도
- Stage 1-2: 2~3자 단어
- Stage 3-4: 3~4자 단어
- Stage 5+: 4자 이상 단어

---

## 4. 혼동 문자 구별 (Confusing Pairs)

### 형식
비슷한 모양의 문자 쌍을 구별하는 연습

### 히라가나 혼동 쌍
- は / ほ / ま
- あ / お
- き / さ
- ぬ / め
- る / ろ
- ね / れ / わ
- い / り
- う / つ / ら

### 카타카나 혼동 쌍
- ツ / シ
- ソ / ン
- ウ / ワ / フ
- ア / マ
- ク / タ / ケ
- ヌ / ス
- コ / ユ
- ナ / メ

### 진행
1. 문자 하나 표시 + "이것은 어떤 문자?" (예: シ)
2. 2지선다: シ(shi) / ツ(tsu)
3. 오답 시 두 문자의 차이점 시각적으로 설명

---

## 5. 받아쓰기 (Dictation)

### 형식
소리 설명을 보고 올바른 가나 순서 맞추기 (Stage 4부터)

### 진행
1. 로마자로 단어 표시 (예: "sakura")
2. 학습자가 가나로 직접 입력: さくら
3. 글자별로 채점하여 틀린 부분 표시

---

## 6. 행(段) 완성 (Row Completion)

### 형식
가나 행의 빈칸 채우기

### 진행
1. 행 일부를 표시 (예: あ _ う _ お)
2. 빈칸에 들어갈 문자 선택
3. 전체 행을 소리내어 읽는 연습 유도

---

## 7. 플래시카드 (Flashcard)

### 형식
`templates/kana-chart.html`을 활용한 인터랙티브 카드

### 앞면
가나 문자 1자 (큰 글씨)

### 뒷면
- 발음 (로마자)
- 한국어 유사 발음
- 소속 행/단
- 핵심 단어 1~2개
- 자기 평가 (알겠어요/애매해요/모르겠어요)

---

## 출제 규칙

1. **현재 스테이지 + 이전 스테이지** 문자를 혼합 출제 (비율 7:3)
2. 오답이 많은 문자는 같은 세션 내에서 **재출제**
3. 혼동 쌍에 속하는 문자는 반드시 쌍으로 묶어서 연습 기회 제공
4. 1라운드 4문제, 라운드 후 오답 복습
5. Stage 1에서는 문자 인식(유형 1)과 발음→문자(유형 2)만 사용
6. Stage 2부터 단어 읽기(유형 3) 추가
7. Stage 4부터 받아쓰기(유형 5) 추가
