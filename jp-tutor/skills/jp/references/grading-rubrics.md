# 채점 기준 (Grading Rubrics)

## 1. 객관식 채점 (퀴즈/플래시카드)

### 판정
- **정답**: correct +1, attempts +1
- **오답**: attempts +1 (correct 변동 없음)

### 배지 계산
`ratio = correct / attempts`

| 비율 | 배지 | 의미 |
|------|------|------|
| >= 80% | 🟦 파랑 | 마스터 |
| >= 60% | 🟩 초록 | 숙련 |
| >= 40% | 🟨 노랑 | 학습중 |
| < 40% | 🟥 빨강 | 보강 필요 |
| 0 attempts | ⬜ 미학습 | 테스트 전 |

- 3회 미만 시도: 임시 배지 (괄호 표시, 예: (🟦))
- 3회 이상 시도: 확정 배지

## 2. 작문 채점 (Writing)

### 평가 항목

| 항목 | 비중 | 평가 내용 |
|------|------|-----------|
| 문법 정확성 | 40% | 조사, 활용, 시제, 접속 오류 |
| 어휘 적절성 | 30% | 레벨에 맞는 단어, 콜로케이션 |
| 자연스러움 | 30% | 일본어다운 어순, 표현, 문체 통일 |

### 등급 기준

| 등급 | 기준 |
|------|------|
| A (우수) | 오류 거의 없음. 자연스럽고 레벨에 맞는 다양한 표현 사용 |
| B (양호) | 소수 오류 있으나 의미 전달에 문제 없음. 기본적으로 자연스러운 표현 |
| C (보통) | 여러 오류가 있어 부분적으로 의미가 불분명. 직역체 빈번 |
| D (노력 필요) | 기본 문법 오류 다수. 의미 전달이 어려운 부분 존재 |

### 피드백 형식
1. 전체 등급과 항목별 점수
2. 오류 목록: 원문 → 수정 → 설명
3. 잘 쓴 표현 1~2개 칭찬
4. 모범 답안 제시

---

## 3. 회화 채점 (Conversation)

### 평가 항목

| 항목 | 비중 | 평가 내용 |
|------|------|-----------|
| 상황 대처 | 40% | 상황에 맞는 적절한 대응, 목표 달성 여부 |
| 경어 적절성 | 30% | 상대방에 맞는 경어 수준, 경어 정확성 |
| 표현 다양성 | 30% | 다양한 문형/어휘 사용, 패턴 반복 회피 |

### 등급 기준

| 등급 | 기준 |
|------|------|
| A (우수) | 상황에 완벽히 대처. 적절한 경어와 다양한 표현 |
| B (양호) | 대체로 적절한 대처. 경어 소수 오류. 기본 표현 활용 |
| C (보통) | 상황 대처 가능하나 어색함. 경어 혼동. 표현 단조로움 |
| D (노력 필요) | 상황 대처 어려움. 경어 오류 다수. 최소한의 표현만 사용 |

### 피드백 형식
1. 전체 등급과 항목별 점수
2. 각 발화별 교정 (원문 → 수정 → 설명)
3. 대안 표현 제안
4. 잘한 점 1~2개 칭찬

---

## 4. 번역 채점 (Translation)

### 평가 항목

| 항목 | 비중 | 평가 내용 |
|------|------|-----------|
| 정확성 | 40% | 원문 의미의 정확한 전달, 누락/오역 여부 |
| 자연스러움 | 30% | 도착어에서 자연스러운 표현, 직역체 회피 |
| 어휘 선택 | 30% | 문맥에 맞는 단어 선택, 뉘앙스 반영 |

### 등급 기준

| 등급 | 기준 |
|------|------|
| A (우수) | 의미 정확하고 자연스러운 번역. 뉘앙스까지 반영 |
| B (양호) | 의미 대체로 정확. 일부 어색한 표현 있으나 이해 가능 |
| C (보통) | 핵심 의미는 전달되나 오역/누락 일부 존재. 직역체 빈번 |
| D (노력 필요) | 의미 전달 불충분. 주요 오역 존재 |

---

## 5. 등급 → 배지 매핑

주관식 평가의 등급은 다음과 같이 배지로 변환됩니다:

| 등급 | 배지 |
|------|------|
| A | 🟦 파랑 |
| B | 🟩 초록 |
| C | 🟨 노랑 |
| D | 🟥 빨강 |

주관식 세션 결과는 해당 카테고리(writing/conversation/translation)의 sessions 배열에 기록됩니다.
관련 개념(사용된 문법/어휘)의 마스터리에도 반영합니다.
