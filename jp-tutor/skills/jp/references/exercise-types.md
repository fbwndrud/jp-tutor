# 학습 모드별 운영 규칙

## 1. 어휘 퀴즈 (Vocabulary Quiz)

### 형식
- 4지선다 객관식
- 1라운드 4문제
- 의미/읽기/용법 변형을 섞어 출제

### 진행 순서
1. 학습자의 레벨과 프로필에 맞는 단어 선택
2. 복습 대상 단어 우선 출제
3. 새 단어는 복습 단어와 섞어 출제 (비율 약 3:1)
4. 퀴즈 규칙(`quiz-rules.md`)에 따라 진행
5. 라운드 후 progress.json의 vocab.concepts 업데이트

---

## 2. 문법 연습 (Grammar Practice)

### 형식
문법 포인트 1개를 선택하여 다음 순서로 진행:

1. **설명**: 문법 포인트의 의미, 접속 규칙, 예문 3개
2. **드릴**: 4문제 연습 (아래 유형을 혼합)
   - 변환 문제: 주어진 문장을 지정 형태로 변환
   - 선택 문제: 빈칸에 알맞은 표현 고르기
   - 오류 찾기: 틀린 부분 찾아 고치기
   - 작문 문제: 주어진 조건으로 문장 만들기

### 난이도 조절
- N5~N4: 단일 문법 포인트, 짧은 문장
- N3: 복합 문법, 중간 길이 문장
- N2~N1: 문어체 포함, 긴 문장, 유사 문법 비교

## 3. 한자 학습 (Kanji Learning)

### 플래시카드 형식
각 한자에 대해 다음 정보를 카드로 제시:

```
━━━━━━━━━━━━━━
    食
━━━━━━━━━━━━━━
음독: ショク, ジキ
훈독: た(べる), く(う)
의미: 먹다, 음식
부수: 食 (식)
획수: 9획
관련 단어: 食事(しょくじ), 食べ物(たべもの), 和食(わしょく)
유사 한자: 飲, 飯, 餅
━━━━━━━━━━━━━━
```

### 진행 순서
1. 플래시카드로 한자 3~5개 학습
2. 학습한 한자로 퀴즈 (quiz-rules.md 적용)
3. 틀린 한자는 세션 마지막에 한 번 더 복습

## 4. 독해 연습 (Reading Comprehension)

### 진행 순서
1. **지문 생성**: 학습자 레벨에 맞는 일본어 지문 생성 (100~400자)
   - N5: 자기소개, 일상 묘사 (100~150자)
   - N4: 편지, 안내문 (150~200자)
   - N3: 에세이, 뉴스 요약 (200~300자)
   - N2: 신문 기사, 평론 (300~400자)
   - N1: 학술문, 문학 (400자+)
2. **내용 이해 질문**: 3~5문제 (선다형 + 주관식 혼합)
3. **어휘/문법 해설**: 지문에 등장한 주요 단어와 문법 포인트 정리

### 주의사항
- N4 이하는 후리가나 필수
- 지문 주제는 학습자 관심사 반영 (가능한 경우)

---

## 5. 작문 연습 (Writing Practice)

### 진행 순서
1. **주제 제시**: 레벨에 맞는 작문 주제 제공
   - N5: 자기소개, 취미 소개 (3~5문장)
   - N4: 일기, 짧은 편지 (5~8문장)
   - N3: 의견문, 경험담 (8~12문장)
   - N2~N1: 에세이, 논설문 (15문장+)
2. **학습자 작성**: 학습자가 일본어로 작문
3. **첨삭**: 채점 기준(grading-rubrics.md)에 따라 평가
   - 문법 정확성 (40%): 문법 오류 지적 및 수정
   - 어휘 적절성 (30%): 더 자연스러운 단어 제안
   - 자연스러움 (30%): 일본어다운 표현으로 개선 제안
4. **모범 답안**: 같은 내용을 자연스러운 일본어로 재작성하여 비교

---

## 6. 번역 연습 (Translation)

### 진행 순서
1. **원문 제시**: 한국어 또는 일본어 문장/단락 제공
2. **학습자 번역**: 학습자가 번역 작성
3. **모범 번역 비교**: 두 가지 모범 번역 제공
   - 직역: 원문 구조에 충실한 번역
   - 의역: 도착어에 자연스러운 번역
4. **채점**: grading-rubrics.md 기준 적용
   - 정확성 (40%): 의미 전달의 정확도
   - 자연스러움 (30%): 도착어에서의 자연스러움
   - 어휘 선택 (30%): 적절한 단어/표현 사용

### 난이도
- 쉬움: 단문 (1문장)
- 보통: 복문 (2~3문장)
- 어려움: 단락 (4문장+), 관용구/속담 포함

---

## 7. 회화 시뮬레이션 (Conversation Simulation)

### 진행 순서
1. **상황 설정**: 구체적인 회화 상황 제시
   - 장소, 상대방, 목적, 사용해야 할 표현/문법 안내
2. **롤플레이**: 4~6턴 대화 진행
   - 튜터가 상대방 역할 수행
   - 대화 중에는 **교정하지 않음** (몰입 유지)
   - 자연스러운 대화 흐름 유지
3. **사후 교정**: 대화 종료 후 피드백
   - 각 발화에 대한 문법/표현 교정
   - 더 자연스러운 대안 표현 제안
   - grading-rubrics.md 기준으로 채점

### 상황 예시
- N5: 자기소개, 물건 사기, 길 묻기
- N4: 약속 잡기, 병원에서, 주문하기
- N3: 상담, 불만 표현, 여행 계획
- N2~N1: 면접, 프레젠테이션, 토론
- 비즈니스: 전화 응대, 회의, 접대

---

## 8. 문화/관용표현 학습 (Culture/Idioms)

### 진행 순서
1. **문화 배경 설명**: 일본 문화의 특정 측면을 한국어로 설명
   - 한국 문화와 비교하여 이해를 돕습니다
2. **관련 표현/관용구 학습**: 3~5개 표현 소개
   - 일본어 표현 + 직역 + 실제 의미 + 사용 예문
3. **퀴즈**: 배운 표현으로 짧은 퀴즈 (quiz-rules.md 적용)

### 주제 예시
- 인사/예절: お辞儀, いただきます, お疲れ様
- 계절: 花見, お中元, 年賀状
- 사회: 空気を読む, 本音と建前, 和
- 비즈니스: 報連相, 根回し, 名刺交換
