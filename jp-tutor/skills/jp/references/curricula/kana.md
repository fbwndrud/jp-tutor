# 가나 커리큘럼 (Pre-N5)

히라가나/카타카나를 처음부터 배우는 완전 초보자를 위한 커리큘럼.
빈도 기반 + 단어 형성 가능성 우선 순서로 구성되어 있다.

## 히라가나 (7단계)

### Stage 1: 모음 + か행 (10자)
あ い う え お か き く け こ

**핵심 단어**: あい(사랑), いか(오징어), あき(가을), かく(쓰다), いく(가다), あう(만나다), おか(언덕), えき(역)

### Stage 2: さ행 + た행 (10자)
さ し す せ そ た ち つ て と

**핵심 단어**: した(아래), すし(초밥), くつ(신발), そと(바깥), かさ(우산), つき(달), しかし(그러나), たかい(높다)

### Stage 3: な행 + は행 (10자)
な に ぬ ね の は ひ ふ へ ほ

**핵심 단어**: なに(무엇), はな(꽃), ほね(뼈), ふね(배), にほん(일본), はし(젓가락), ほし(별), ひと(사람)

### Stage 4: ま행 + や행 (8자)
ま み む め も や ゆ よ

**핵심 단어**: やま(산), もの(것), ゆめ(꿈), まゆ(눈썹), よむ(읽다), みせ(가게), もも(복숭아), やすみ(휴일)

### Stage 5: ら행 + わ행 + ん (8자)
ら り る れ ろ わ を ん

**핵심 단어**: さくら(벚꽃), りんご(사과), みる(보다), われる(깨지다), ほん(책), せんせい(선생님), にほんご(일본어)

> Stage 5 완료 후 카타카나 병행 학습 시작

### Stage 6: 탁음 + 반탁음 (25자)
が ぎ ぐ げ ご ざ じ ず ぜ ぞ だ ぢ づ で ど ば び ぶ べ ぼ ぱ ぴ ぷ ぺ ぽ

**핵심 단어**: がっこう(학교), じかん(시간), でんわ(전화), ぶんか(문화), ぜんぶ(전부), でも(그래도), だから(그래서), ぱん(빵)

### Stage 7: 요음 (33자)
きゃ きゅ きょ しゃ しゅ しょ ちゃ ちゅ ちょ にゃ にゅ にょ ひゃ ひゅ ひょ みゃ みゅ みょ りゃ りゅ りょ
ぎゃ ぎゅ ぎょ じゃ じゅ じょ びゃ びゅ びょ ぴゃ ぴゅ ぴょ

**핵심 단어**: きょう(오늘), しゅくだい(숙제), ちゃ(차), びょういん(병원), りょこう(여행), じゅう(열), しゃしん(사진)

## 카타카나 (4단계)

> 히라가나 Stage 5 이후 병행 학습

### K1: ア행~タ행 (20자)
ア イ ウ エ オ カ キ ク ケ コ サ シ ス セ ソ タ チ ツ テ ト

**핵심 단어**: コーヒー(커피), タクシー(택시), テスト(테스트), カメラ(카메라), サイト(사이트), スキー(스키)

### K2: ナ행~ヤ행 (18자)
ナ ニ ヌ ネ ノ ハ ヒ フ ヘ ホ マ ミ ム メ モ ヤ ユ ヨ

**핵심 단어**: ハンバーガー(햄버거), メニュー(메뉴), ホテル(호텔), ノート(노트), ニュース(뉴스)

### K3: ラ행~ン + 탁음/반탁음 (28자)
ラ リ ル レ ロ ワ ヲ ン
ガ ギ グ ゲ ゴ ザ ジ ズ ゼ ゾ ダ ヂ ヅ デ ド バ ビ ブ ベ ボ

**핵심 단어**: レストラン(레스토랑), ゲーム(게임), バス(버스), ドア(문), ビール(맥주), パン(빵)

### K4: 요음 + 특수 표기 (36자+)
キャ キュ キョ シャ シュ ショ チャ チュ チョ ニャ ニュ ニョ ヒャ ヒュ ヒョ ミャ ミュ ミョ リャ リュ リョ
ギャ ギュ ギョ ジャ ジュ ジョ ビャ ビュ ビョ ピャ ピュ ピョ
+ 특수: ティ ディ ファ フィ フェ フォ ヴァ ヴィ ヴ ヴェ ヴォ

**핵심 단어**: チョコレート(초콜릿), シャツ(셔츠), パーティー(파티), ファイル(파일), ヴァイオリン(바이올린)

## 졸업 기준

| 조건 | 기준 |
|------|------|
| 히라가나 정확도 | >= 95% (전체 청음 46자 기준) |
| 카타카나 정확도 | >= 90% (전체 청음 46자 기준) |
| 탁음/반탁음 | >= 85% |
| 요음 | >= 80% |
| 최소 시도 | 각 문자 3회 이상 |

모든 조건 충족 시 **Pre-N5 → N5 자동 승급** 안내.

## 학습 팁

- 히라가나/카타카나는 **소리**와 **모양**을 동시에 익히는 것이 핵심
- 한 글자씩 외우지 말고, **단어 속에서** 자연스럽게 반복
- 행(あ행, か행...)별로 그룹 학습이 효과적
- 비슷한 모양의 문자는 비교하며 학습 (예: は/ほ, ツ/シ, ソ/ン)
- 매일 5~10분 꾸준히 > 한번에 1시간
