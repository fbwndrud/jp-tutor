# 비즈니스 일본어 커리큘럼

## 모듈 1: 비즈니스 매너 기초

### 학습 목표
기본적인 비즈니스 인사, 자기소개, 명함 교환, 기초 경어를 익힌다.

### 내용
1. **인사 표현**
   - 아침/낮/저녁 인사 (おはようございます, お疲れ様です, お先に失礼します)
   - 외출/귀사 인사 (行ってまいります, ただいま戻りました)
   - 계절 인사 (時候の挨拶)

2. **자기소개**
   - 회사명/부서/이름 소개 패턴
   - 입사 인사, 부임 인사
   - 간단한 경력 소개

3. **명함 교환 (名刺交換)**
   - 교환 순서와 매너
   - 주고받을 때 표현
   - 명함 취급 예절

4. **기초 경어**
   - 존경어 (いらっしゃる, おっしゃる, ご覧になる)
   - 겸양어 (参る, 申す, 拝見する)
   - 정중어 (です/ます, ございます)
   - 이중경어 주의 사항

## 모듈 2: 비즈니스 이메일

### 학습 목표
비즈니스 이메일의 기본 구조와 상황별 작성법을 익힌다.

### 내용
1. **이메일 기본 구조**
   - 제목 작성법
   - 서두 인사 (お世話になっております)
   - 본문 구성
   - 맺음말 (何卒よろしくお願いいたします)
   - 서명

2. **상황별 이메일**
   - 의뢰/요청 (お願いメール)
   - 사과 (お詫びメール)
   - 보고 (報告メール)
   - 문의 (問い合わせメール)
   - 일정 조율 (日程調整)
   - 감사 (お礼メール)

3. **이메일 에티켓**
   - CC/BCC 사용법
   - 첨부파일 언급
   - 답장 속도와 예절
   - 사내 vs 사외 이메일 차이

## 모듈 3: 회의/보고

### 학습 목표
보고, 연락, 상담(報連相)의 기본과 회의 진행/참여 표현을 익힌다.

### 내용
1. **보·련·상 (報連相)**
   - 보고(報告): 결과 보고, 중간 보고, 문제 보고
   - 연락(連絡): 정보 공유, 일정 변경, 긴급 연락
   - 상담(相談): 판단을 구할 때, 의견을 구할 때

2. **회의 진행**
   - 개회/폐회 표현
   - 안건 소개, 의견 요청
   - 찬성/반대 표현
   - 정리/결론 도출

3. **프레젠테이션**
   - 도입: 자기소개, 발표 개요
   - 본론: 데이터 설명, 그래프 설명
   - 결론: 요약, 질의응답
   - 자주 쓰는 표현 패턴

4. **의견 표현**
   - 정중하게 의견 말하기 (~と考えております)
   - 반대 의견 부드럽게 전달 (~ではないかと思います)
   - 제안하기 (~てはいかがでしょうか)

## 모듈 4: 전화 응대

### 학습 목표
비즈니스 전화의 기본 매너와 상황별 대응법을 익힌다.

### 내용
1. **전화 받기**
   - 기본 응대 (はい、○○でございます)
   - 상대방 확인
   - 담당자 연결 (少々お待ちください)
   - 대기/보류 표현

2. **전화 걸기**
   - 자기소개 + 용건 전달
   - 담당자 요청 (~様はいらっしゃいますか)
   - 부재 시 대응 (伝言をお願いしてもよろしいでしょうか)

3. **부재 대응**
   - 외출 중/회의 중/퇴근 안내
   - 메모 남기기
   - 콜백 요청/약속

4. **클레임 대응**
   - 불만 접수 표현
   - 사과와 대응 안내
   - 에스컬레이션 표현

## 모듈 5: 업종별 용어

### 학습 목표
자신의 업종에 맞는 전문 일본어를 학습한다.

### 기본 제공 업종

1. **IT/테크**
   - 개발 용어 (要件定義, 設計, 実装, テスト, リリース)
   - 회의 용어 (スプリント, バックログ, レビュー)
   - 장애 대응 (障害, 復旧, 原因調査)

2. **제조업**
   - 생산 용어 (製造, 品質管理, 検査, 出荷)
   - 거래 용어 (見積, 発注, 納品, 請求)
   - 품질 관련 (不良品, クレーム対応, 改善)

3. **금융**
   - 은행 용어 (口座, 振込, 融資, 為替)
   - 투자 용어 (株式, 債券, 利回り, リスク)
   - 보험 용어 (保険料, 保障, 契約, 解約)

4. **사용자 맞춤**
   - 학습자가 자신의 업종을 지정하면 해당 분야 용어를 학습
   - 실제 업무 자료(이메일, 문서 등)를 기반으로 학습 가능

## 모듈 6: 고급 비즈니스

### 학습 목표
협상, 계약, 접대, 출장 등 고급 비즈니스 상황을 다룬다.

### 내용
1. **협상 (交渉)**
   - 조건 제시/협의 표현
   - 양보/타협 표현
   - 합의 도달 표현
   - 결렬 시 표현

2. **계약 (契約)**
   - 계약서 기본 용어
   - 조건 확인/수정 요청
   - 계약 체결/해지 표현
   - 기밀유지(NDA) 관련

3. **접대 (接待)**
   - 식사 접대 매너
   - 술자리 문화 (飲み会)
   - 건배 인사, 주문, 계산
   - 감사 인사 (後日のお礼)

4. **출장 (出張)**
   - 출장 준비 (手配, 予約)
   - 방문 인사 (アポイント)
   - 공장/사무실 견학
   - 출장 보고서 작성
