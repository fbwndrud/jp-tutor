# jp-tutor

Claude Cowork용 일본어 학습 플러그인. (v0.8.1)

## 학습 프로필

1. **JLPT 독학** - N5~N1 단계별 시험 준비
2. **비즈니스 일본어** - 경어, 이메일, 회의, 전화 등 실무 일본어
3. **교재 기반** - 사용자 교재 PDF/사진을 기반으로 복습 및 연습

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/jp` | 메인 진입점. 자연어로 요청하면 자동 라우팅 |
| `/jp:start` | 초기 설정 (레벨 선택, 프로필 구성) |
| `/jp:daily` | 오늘의 학습 세션 시작 |
| `/jp:review` | 에빙하우스 기반 복습 |
| `/jp:vocab` | 어휘 퀴즈 |
| `/jp:grammar` | 문법 연습 |
| `/jp:kanji` | 한자 학습 |
| `/jp:kana` | 히라가나/가타카나 학습 |
| `/jp:translate` | 번역 연습 |
| `/jp:culture` | 문화/관용표현 학습 |
| `/jp:biz` | 비즈니스 일본어 모드 |
| `/jp:textbook` | 교재 기반 학습 모드 |
| `/jp:game` | 미니게임 (매칭카드, 셔플퍼즐, 스네이크매치 등 10종) |
| `/jp:dashboard` | 학습 현황 대시보드 |
| `/jp:export` | CSV/Anki 내보내기 |
| `/jp:report` | 주간/월간 학습 리포트 |
| `/jp:update` | 진행 데이터 업데이트/동기화 |

## 미니게임

10종의 인터랙티브 HTML 미니게임으로 학습 효과를 극대화합니다.

| 게임 | 설명 |
|------|------|
| 매칭 카드 | 카드 뒤집어 짝 맞추기 |
| 셔플 퍼즐 | 문장 어순 배열 |
| 스네이크 매치 | 그리드에서 짝 연결 |
| 조사 빈칸 | 문장에 알맞은 조사 채우기 |
| 부수 조립 | 부수를 조합해 한자 완성 |
| 듣고 그리기 | 음성 듣고 히라가나/가타카나 따라 쓰기 |
| 끝말잇기 | 일본어 끝말잇기 (대화형) |
| 경어 변환 | 경어 레벨 변환 연습 (대화형) |
| 문맥 추리 | 문맥에서 단어 의미 추리 (대화형) |
| 플래시카드 | SRS 기반 플래시카드 복습 |
