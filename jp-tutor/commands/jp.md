# 일본어 학습 범용 진입점

## 사전 준비

1. 먼저 `skills/jp/SKILL.md` 스킬 파일을 읽어 튜터 역할과 규칙을 숙지하세요.
2. `jp-data/progress.json` 파일을 읽으세요.
   - 파일이 없으면: "아직 초기 설정이 되어 있지 않습니다. `/jp:start`로 시작해주세요!" 라고 안내하고 종료합니다.

## 인자가 없는 경우 (현황 + 추천)

인자 없이 호출된 경우 다음을 수행합니다:

1. **현재 상태 요약**:
   - 프로필, 레벨, XP, 연속 학습일
   - 오늘 복습 대상 개수 (`nextReview <= 오늘` 인 개념 수)
   - 최근 학습 세션 정보

2. **오늘의 추천**:
   - 복습 대상이 있으면: "복습할 항목이 {n}개 있어요! `/jp:review`로 복습하세요."
   - 복습 대상이 없으면: 현재 프로필에 맞는 학습 추천
     - jlpt: 현재 레벨 커리큘럼 기반 다음 학습 단원 추천
     - business: 다음 비즈니스 모듈 추천
     - textbook: 다음 챕터 추천
   - 일일 학습을 아직 안 했으면: "`/jp:daily`로 오늘의 학습을 시작하세요!"

## 인자가 있는 경우 (의도 파악 + 라우팅)

사용자 입력을 분석하여 SKILL.md의 세션 라우팅 규칙에 따라 적절한 모드로 라우팅합니다:

- "히라가나", "카타카나", "가나", "50음도", "kana" → `/jp:kana` 가나 학습 모드 실행
- "단어", "어휘", "vocab" → 어휘 학습 모드 실행
- "문법", "grammar" → 문법 학습 모드 실행
- "한자", "kanji" → 한자 학습 모드 실행
- "읽기", "독해", "reading" → 독해 연습 모드 실행
- "작문", "writing", "일기" → 작문 연습 모드 실행
- "회화", "대화", "conversation" → 회화 연습 모드 실행
- "번역", "translate" → 번역 모드 실행
- "비즈니스", "경어", "이메일", "biz" → 비즈니스 일본어 모드 실행
- "교재", "교과서", "textbook" → 교재 기반 학습 모드 실행
- "복습", "review", "에빙하우스" → `/jp:review` 안내
- "현황", "통계", "대시보드", "dashboard" → `/jp:dashboard` 안내
- "오늘", "공부 시작", "daily" → `/jp:daily` 안내
- 그 외: 현재 프로필과 최근 학습 이력을 기반으로 가장 적절한 모드를 추천

## 라우팅 후 실행

해당 모드를 직접 실행합니다:
1. 필요한 참조 파일을 읽습니다 (quiz-rules.md, exercise-types.md 등)
2. progress.json에서 해당 카테고리의 현재 상태를 파악합니다
3. 적절한 난이도의 학습/연습을 제공합니다
4. 세션 완료 후 progress.json을 업데이트합니다 (attempts, correct, lastReview, nextReview, XP, streak)

## 언어 규칙

- 모든 설명과 피드백은 **한국어**로 제공합니다.
- 예문은 **일본어** 원문 + 한국어 번역을 병기합니다.
- N4 이하 학습자에게는 반드시 후리가나를 병기합니다.
