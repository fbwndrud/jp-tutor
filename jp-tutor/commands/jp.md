# /jp -- 일본어 학습 튜터

사용자가 일본어 학습을 요청했습니다.

사용자 입력: $ARGUMENTS

스킬 `japanese-tutor`의 SKILL.md 가이드라인을 따라 세션을 진행하세요.

## 사용 예시

```
/jp                                    # 상태 기반 자동 추천
/jp N4 문법 퀴즈 내줘                    # 문법 퀴즈
/jp 한자 읽기 연습                       # 한자 학습
/jp 편의점에서 물건 사는 상황으로 회화 연습  # 회화 시뮬레이션
/jp 진도 보여줘                         # 대시보드
```
