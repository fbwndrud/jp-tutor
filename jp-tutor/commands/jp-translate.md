# 번역 연습 (jp:translate)

## 사전 확인

1. `jp-data/progress.json` 파일을 읽습니다.
2. 파일이 없으면 "아직 학습 프로필이 없습니다. `/jp:start`로 먼저 초기 설정을 해주세요."라고 안내하고 종료합니다.

## 설정 결정

- **방향**: 인자로 지정되면 해당 방향, 없으면 학습자에게 물어봅니다.
  - `한일` (Korean -> Japanese): 한국어 문장을 일본어로 번역
  - `일한` (Japanese -> Korean): 일본어 문장을 한국어로 번역
- **레벨**: 인자로 지정되면 해당 레벨, 없으면 `progress.json`의 `level` 값 사용.
- **주제**: 인자로 지정되면 해당 주제, 없으면 레벨에 맞는 일상 주제 자동 선택.

## 문제 출제

학습자 레벨에 맞는 **3~5개 문장**을 제시합니다.

### 난이도별 문장 길이
- N5~N4: 단문 1문장씩
- N3: 복문 (2~3문장 연결)
- N2~N1: 단락 (4문장+), 관용구/속담 포함 가능

### 진행 방식
한 문장씩 순서대로 진행합니다:
1. 원문을 제시합니다.
2. 학습자의 번역을 기다립니다.
3. 피드백을 제공합니다.
4. 다음 문장으로 넘어갑니다.

## 피드백

`skills/jp/references/grading-rubrics.md`의 번역 채점 기준을 적용합니다.

각 문장 번역 후 다음을 제공합니다:

### 1. 비교
```
📝 번역 피드백
━━━━━━━━━━━━━━
원문: {원문}
학습자 번역: {학습자 답}
━━━━━━━━━━━━━━
```

### 2. 모범 번역 (두 가지)
- **직역**: 원문 구조에 충실한 번역
- **의역**: 도착어에서 자연스러운 번역

### 3. 표현 선택 설명
- 학습자가 사용한 표현과 모범 답안의 차이를 설명합니다.
- 왜 특정 표현이 더 적절한지 이유를 제시합니다.

### 4. 부자연스러운 표현 지적
- 직역체나 부자연스러운 표현을 구체적으로 지적합니다.
- 대안 표현을 제시합니다.

## 세션 종료 후

### 전체 평가
채점 기준에 따라 종합 등급을 부여합니다:

| 항목 | 비중 | 설명 |
|------|------|------|
| 정확성 | 40% | 원문 의미의 정확한 전달 |
| 자연스러움 | 30% | 도착어에서의 자연스러운 표현 |
| 어휘 선택 | 30% | 문맥에 맞는 단어 선택 |

```
📊 번역 연습 결과
━━━━━━━━━━━━━━
방향: {한일/일한}
문장 수: {count}개
종합 등급: {A/B/C/D}
- 정확성: {점수}
- 자연스러움: {점수}
- 어휘 선택: {점수}
획득 XP: +{xp}
━━━━━━━━━━━━━━
```

### progress.json 업데이트
- `translation.sessions` 배열에 세션 기록 추가:
  - `date`, `direction`, `level`, `grade`, `sentenceCount`
- 사용된 문법/어휘의 마스터리에도 반영합니다.
- XP 부여: 번역 세션 완료 +20
- 등급 -> 배지 매핑: A->🟦, B->🟩, C->🟨, D->🟥
- 스트릭 업데이트

## 언어 규칙
- 모든 설명과 피드백은 **한국어**로 제공합니다.
- N4 이하 학습자에게는 일본어 원문에 **후리가나**를 병기합니다.
- 오류 설명 시 틀린 부분을 구체적으로 지적하고 올바른 표현과 이유를 설명합니다.
