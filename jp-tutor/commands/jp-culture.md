# 문화/관용표현 학습 (jp:culture)

## 사전 확인

1. `jp-data/progress.json` 파일을 읽습니다.
2. 파일이 없으면 "아직 학습 프로필이 없습니다. `/jp:start`로 먼저 초기 설정을 해주세요."라고 안내하고 종료합니다.

## 카테고리 결정

인자로 지정되면 해당 카테고리, 없으면 목록을 보여주고 선택하게 합니다:

```
🎌 문화/관용표현 카테고리
━━━━━━━━━━━━━━━━━━
1. 인사 - 인사말과 예절 표현
2. 비즈니스 - 직장/비즈니스 표현과 문화
3. 일상 - 일상생활 속 표현과 관습
4. 계절 - 계절 행사와 관련 표현
5. 관용구 - 관용구와 속담
━━━━━━━━━━━━━━━━━━
```

## 문화 배경 설명

선택한 카테고리의 일본 문화 측면을 **한국어**로 설명합니다.
- 한국 문화와 비교하여 이해를 돕습니다.
- 왜 이런 표현/관습이 생겼는지 배경을 설명합니다.

## 관련 표현 학습

**3~5개** 표현/관용구를 소개합니다. 각 표현에 대해:

```
━━━━━━━━━━━━━━━━━━
표현: お疲(つか)れ様(さま)でした
읽기: おつかれさまでした
직역: 수고하셨습니다
실제 의미: 업무/활동 종료 시 상대의 노고를 치하하는 인사
사용 예문:
  A: 今日(きょう)も一日(いちにち)お疲(つか)れ様(さま)でした。
  → 오늘도 하루 수고하셨습니다.
한국어 대응: "수고하셨습니다" (가장 가까운 표현)
━━━━━━━━━━━━━━━━━━
```

### 카테고리별 주제 예시
- **인사**: お辞儀, いただきます, ごちそうさま, お疲れ様, よろしくお願いします
- **비즈니스**: 報連相, 根回し, 名刺交換, お世話になっております, 失礼いたします
- **일상**: 空気を読む, 本音と建前, お邪魔します, つまらないものですが
- **계절**: 花見, お中元, お歳暮, 年賀状, 衣替え
- **관용구**: 猫の手も借りたい, 石の上にも三年, 花より団子, 七転び八起き

## 확인 퀴즈

`skills/jp/references/quiz-rules.md` 규칙을 따릅니다.

### 구성
- **2~3문제**, 상황 적합 표현 선택형
- 배운 표현 범위 내에서 출제

### 문제 형식
상황을 제시하고 가장 적절한 표현을 고르는 형식:

```
Q. 회사에서 퇴근할 때 동료에게 하는 인사로 가장 적절한 것은?
A. いただきます
B. お疲れ様でした
C. ごちそうさまでした
D. よろしくお願いします
```

### 즉시 피드백
각 문제 답변 직후 정답/오답 표시 및 해설을 제공합니다.
- 왜 그 상황에서 해당 표현이 적절한지 설명합니다.

## 세션 종료 후

### 결과 요약
```
🎌 문화/관용표현 학습 결과
━━━━━━━━━━━━━━━━━━
카테고리: {카테고리}
학습 표현: {표현 목록}
퀴즈 점수: {correct}/{total}
획득 XP: +{xp}
━━━━━━━━━━━━━━━━━━
```

### progress.json 업데이트
- 학습한 표현을 `vocab.concepts`에 추가합니다:
  - 정답: `correct` +1, `attempts` +1
  - 오답: `attempts` +1
  - `lastReview`, `nextReview` 에빙하우스 간격에 따라 갱신
- XP 부여: 문화 학습 완료 +15, 퀴즈 정답당 +10, 오답 +2
- 스트릭 업데이트

## 언어 규칙
- 모든 설명과 피드백은 **한국어**로 제공합니다.
- N4 이하 학습자에게는 일본어 표현에 반드시 **후리가나**를 병기합니다.
- 문화 비교 시 한국 문화를 기준으로 설명하여 이해를 돕습니다.
