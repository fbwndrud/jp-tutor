# 히라가나/카타카나 학습

## 사전 준비

1. `skills/jp/SKILL.md` 스킬 파일을 읽어 튜터 역할을 숙지하세요. (특히 TTS 섹션 확인)
2. TTS 환경을 감지합니다 (SKILL.md TTS 섹션 참조). Cowork/원격 환경이면 HTML TTS 플레이어를 생성합니다.
3. `jp-data/progress.json`을 읽으세요.
   - 파일이 없으면: "아직 초기 설정이 되어 있지 않습니다. `/jp:start`로 시작해주세요!" 안내 후 종료.
   - level이 "Pre-N5"가 아니면: 이미 가나를 졸업한 학습자입니다. "가나 복습이 필요하시면 말씀해주세요!" 라고 안내하되, 원하면 진행합니다.
3. `skills/jp/references/curricula/kana.md`를 읽어 커리큘럼을 숙지하세요.
4. `skills/jp/references/exercise-types-kana.md`를 읽어 연습 유형을 숙지하세요.

## 현재 진행 단계 파악

progress.json의 `kana` 필드에서 현재 상태를 확인합니다:

1. `kana.hiraStage`: 현재 히라가나 단계 (1~7)
2. `kana.kataStage`: 현재 카타카나 단계 (0~4, 0이면 아직 미시작)
3. `kana.concepts`: 각 문자별 학습 데이터

카타카나는 히라가나 Stage 5 완료 후 자동 시작됩니다.

## 모드별 진행

### 인자 없음 또는 "stage" → 자동 진행

현재 단계에 맞는 학습 세트를 자동으로 구성합니다:

1. **새 문자 소개** (3~5자)
   - 큰 글씨로 문자 표시
   - 발음 (로마자 + 한국어 유사음)
   - 획순 설명 (간단히)
   - 핵심 단어 1~2개와 함께 제시
   - 비슷한 모양의 문자가 있으면 차이점 설명

2. **발음 듣기 (TTS)** — 퀴즈 전 필수 단계
   - 소개한 문자와 단어를 모아 TTS 플레이어 HTML을 **먼저** 생성합니다.
   - Cowork/원격: `jp-data/tts-player.html` 생성 후 "발음을 먼저 들어보세요!" 안내
   - 로컬 macOS: `say -v Kyoko` 로 각 문자 직접 재생
   - 유저가 충분히 듣고 학습한 뒤 퀴즈로 넘어갑니다.

3. **미니게임으로 연습** — 퀴즈 전 필수 단계
   - 소개한 문자들로 미니게임 1판을 생성하여 연습합니다:
     - 매칭카드: 히라가나↔로마자 또는 히라가나↔한국어 발음 매칭
     - 듣고 그리기 (Stage 2+, 히라가나 20자 이상 학습 시): 음성 듣고 따라 쓰기
   - 게임 티어는 1(입문)로 고정, 시간 넉넉하게 설정
   - "게임으로 먼저 연습해볼까요?" 안내 후 진행

4. **연습 퀴즈** (8~12문제)
   - 현재 스테이지 문자 70% + 이전 복습 30%
   - exercise-types-kana.md의 유형을 스테이지에 맞게 혼합
   - 오답 문자는 세션 마지막에 재출제

5. **세션 마무리**
   - 이번 세션 결과 요약
   - 다음에 배울 문자 미리보기
   - 스테이지 진급 조건 안내

### "chart" → 50음도 차트

1. `skills/jp/references/templates/kana-chart.html` 파일이 있는지 확인합니다.
2. 있으면:
   - progress.json의 kana.concepts에서 각 문자별 마스터리 데이터를 수집
   - 템플릿의 `const DATA = null;`을 실제 데이터로 치환

### "quiz" → 복습 퀴즈

기존에 학습한 모든 문자에서 복습 퀴즈를 출제합니다:

## progress.json 업데이트

세션 완료 후:

1. **각 문자별**: kana.concepts의 `{문자}_hira` 또는 `{문자}_kata` 키 업데이트
   - `attempts` +1, 정답 시 `correct` +1
   - `lastReview` = 오늘, `nextReview` = 에빙하우스 간격 적용
2. **스테이지 진급 판단**:
   - 현재 스테이지 모든 문자의 정답률 >= 80% + 최소 3회 시도 → 다음 스테이지로
   - `kana.hiraStage` 또는 `kana.kataStage` 업데이트
3. **N5 졸업 판단**:
   - 히라가나 정확도 >= 95% (청음 46자)
   - 카타카나 정확도 >= 90% (청음 46자)
   - 탁음/반탁음 >= 85%, 요음 >= 80%
   - 조건 충족 시: "축하합니다! 가나를 마스터했습니다! N5 레벨로 승급합니다!" 메시지
   - progress.json의 `level`을 "N5"로 변경
4. **XP**: 문자 정답 +5, 새 문자 첫 학습 +10, 스테이지 클리어 +50
5. **스트릭/레벨**: SKILL.md 규칙대로 업데이트

## 카타카나 병행 안내

히라가나 Stage 5 완료 시:
```
히라가나 기본 글자를 모두 배웠습니다!
이제부터 카타카나도 함께 학습합니다.

카타카나는 주로 외래어를 표기할 때 사용해요.
히라가나와 같은 발음이지만 모양이 다릅니다.
예: あ(a) = ア(a), か(ka) = カ(ka)
```

## TTS 활용

가나 학습에서 TTS는 특히 중요합니다. **반드시 퀴즈 전에 발음을 먼저 들려줍니다.**

### 로컬 macOS 환경
- `say -v Kyoko -r 100 "あ"` 로 직접 재생
- 새 문자 소개 시 각 문자마다 바로 재생

### Cowork/원격 환경 (기본)
새 문자 소개 직후, 퀴즈 시작 전에 TTS 플레이어를 생성합니다:
1. `skills/jp/references/templates/tts-player.html` 템플릿을 읽습니다.
2. `const DATA = null;`을 실제 데이터로 치환합니다:
   ```json
   {
     "title": "가나 Stage 1 발음",
     "rate": 0.7,
     "words": [
       { "jp": "あ", "reading": "a", "meaning": "아" },
       { "jp": "あい", "reading": "ai", "meaning": "사랑" }
     ]
   }
   ```
3. `jp-data/tts-player.html`로 저장합니다.
4. "발음을 먼저 들어보세요! `jp-data/tts-player.html`을 브라우저에서 열어주세요" 안내합니다.
5. **유저가 발음을 충분히 듣고 준비되면 퀴즈를 시작합니다.**

TTS 플레이어는 음성 선택 (브라우저에 여러 일본어 음성이 있을 경우), 속도 조절, 전체 재생, 반복 재생 기능을 지원합니다.

## 언어 규칙

- 모든 설명은 **한국어**로 제공합니다.
- 발음 표기: 로마자 + 한국어 유사음 병기 (예: か = ka (카))
- 새 문자 소개 시 항상 연관 단어를 함께 제시합니다.
- 격려 메시지를 자연스럽게 포함합니다.
