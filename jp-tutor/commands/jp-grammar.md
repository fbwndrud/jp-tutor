# 문법 연습 (jp:grammar)

## 사전 확인

1. `jp-data/progress.json` 파일을 읽습니다.
2. 파일이 없으면 "아직 학습 프로필이 없습니다. `/jp:start`로 먼저 초기 설정을 해주세요."라고 안내하고 종료합니다.

## 레벨 및 문법 포인트 결정

- **레벨**: 인자로 지정되면 해당 레벨, 없으면 `progress.json`의 `level` 값 사용.
- **문법 포인트**: 인자로 지정되면 해당 문법, 없으면 레벨에 맞는 미학습/취약 문법 자동 선택.

## 신규 vs 복습 판별

`progress.json`의 `grammar.concepts`에서 해당 문법 포인트를 확인합니다:
- **신규** (concepts에 없거나 attempts가 0): 패턴 설명 -> 드릴 순서로 진행
- **복습** (concepts에 존재하고 attempts > 0): 설명 생략, 바로 드릴 진행

## 패턴 설명 (신규 문법인 경우)

`skills/jp/references/exercise-types.md`의 "문법 연습" 섹션을 참조합니다.

다음 구조로 설명합니다:
1. **구조**: 접속 규칙과 형태 (예: 동사 て형 + ください)
2. **의미**: 한국어로 의미 설명
3. **예문**: 2~3개 예문 (일본어 + 한국어 번역)
   - N4 이하는 후리가나 필수

```
📖 문법 포인트: ~てください
━━━━━━━━━━━━━━━━━━━━
구조: 동사 て형 + ください
의미: ~해 주세요 (정중한 요청/부탁)

예문:
1. ここに名前(なまえ)を書(か)いてください。
   → 여기에 이름을 써 주세요.
2. もう少(すこ)し待(ま)ってください。
   → 조금 더 기다려 주세요.
━━━━━━━━━━━━━━━━━━━━
```

## 드릴 (4문제)

문제 유형을 혼합하여 출제합니다:

1. **변환 문제**: 주어진 문장을 지정 형태로 변환 (주관식)
2. **선택 문제**: 빈칸에 알맞은 문법 표현 고르기 (4지선다)
3. **오류 찾기**: 문장에서 문법 오류를 찾아 고치기 (주관식)
4. **작문 문제**: 주어진 조건으로 문장 만들기 (주관식)

### 난이도 조절
- N5~N4: 단일 문법 포인트, 짧은 문장
- N3: 복합 문법, 중간 길이 문장
- N2~N1: 문어체 포함, 긴 문장, 유사 문법 비교

### 피드백
각 문제 답변 직후:
1. 정답/오답 표시 (⭕/❌)
2. 정답 공개 (오답인 경우)
3. 왜 그것이 정답인지 설명
4. 오답 분석 (오답인 경우)

## 라운드 종료 후

### 결과 요약
```
📖 문법 연습 결과
━━━━━━━━━━━━━━
문법 포인트: {문법 포인트}
점수: {correct}/4
1. ⭕/❌ {유형} - {간단 설명}
2. ⭕/❌ {유형} - {간단 설명}
3. ⭕/❌ {유형} - {간단 설명}
4. ⭕/❌ {유형} - {간단 설명}
획득 XP: +{xp}
━━━━━━━━━━━━━━
```

### progress.json 업데이트
- `grammar.concepts`에서 해당 문법 포인트:
  - 정답: `correct` +1, `attempts` +1
  - 오답: `attempts` +1
  - `lastReview`를 오늘 날짜로 갱신
  - `nextReview`를 에빙하우스 간격에 따라 갱신
- XP 부여: 문법 연습 완료 +15, 정답당 추가 +10, 오답 +2
- 스트릭 업데이트

## 언어 규칙
- 모든 설명과 피드백은 **한국어**로 제공합니다.
- 문법 용어는 한국어 우선, 괄호 안에 일본어 문법 용어 병기 (예: 수동형(受身形))
- N4 이하 학습자에게는 반드시 **후리가나**를 병기합니다.
