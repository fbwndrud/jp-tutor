# 어휘 퀴즈 (jp:vocab)

## 사전 확인

1. `jp-data/progress.json` 파일을 읽습니다.
2. 파일이 없으면 "아직 학습 프로필이 없습니다. `/jp:start`로 먼저 초기 설정을 해주세요."라고 안내하고 종료합니다.

## 레벨 및 주제 결정

- **레벨**: 인자로 지정되면 해당 레벨 사용, 없으면 `progress.json`의 `level` 값 사용.
- **주제**: 인자로 지정되면 해당 주제 사용, 없으면 아래 우선순위로 자동 선택:
  1. 아직 테스트하지 않은 단어 (미학습)
  2. 정답률이 낮은 단어 (취약)
  3. 복습 시기가 된 단어 (일반)

## 퀴즈 진행

`skills/jp/references/quiz-rules.md`의 규칙을 반드시 따릅니다.

### 라운드 구성
- 1라운드 = **4문제**, 각 문제 **4지선다** (A, B, C, D)
- 문제 유형을 섞어 출제합니다:
  - 의미 -> 단어 (한국어 뜻을 보고 일본어 단어 고르기)
  - 단어 -> 의미 (일본어 단어를 보고 한국어 뜻 고르기)
  - 읽기 (한자 단어의 올바른 히라가나 읽기 고르기)
  - 용법 (문장 빈칸에 알맞은 단어 고르기)

### 선택지 생성
- **그럴듯한 오답**: 같은 품사, 비슷한 의미 범주 또는 비슷한 발음에서 선택합니다.
- 정답이 명백하게 튀지 않도록 선택지 길이와 형식을 통일합니다.
- 정답 위치는 랜덤으로 배치합니다.

### 무힌트 정책
- 퀴즈 진행 중에는 **절대** 힌트를 제공하지 않습니다.
- 학습자가 힌트를 요청하면 "퀴즈 모드에서는 힌트를 드릴 수 없어요. 최선의 답을 골라보세요!"라고 안내합니다.

### 즉시 피드백
각 문제 답변 직후:
1. 정답이면 ⭕, 오답이면 ❌ 표시
2. 오답인 경우 올바른 답 공개
3. 왜 그것이 정답인지 간단히 설명 (1~2문장)
4. 학습자가 고른 오답이 왜 틀렸는지 설명 (오답인 경우만)

## 라운드 종료 후

### 결과 요약
```
📝 어휘 퀴즈 결과
━━━━━━━━━━━━━━
점수: {correct}/4
1. ⭕/❌ {단어} - {뜻}
2. ⭕/❌ {단어} - {뜻}
3. ⭕/❌ {단어} - {뜻}
4. ⭕/❌ {단어} - {뜻}
획득 XP: +{xp}
━━━━━━━━━━━━━━
```

### progress.json 업데이트
- 각 단어의 `vocab.concepts`에서:
  - 정답: `correct` +1, `attempts` +1
  - 오답: `attempts` +1
  - `lastReview`를 오늘 날짜로 갱신
  - `nextReview`를 에빙하우스 간격(1d->3d->7d->14d->30d->60d)에 따라 갱신
    - 정답이면 다음 간격으로, 오답이면 1d로 리셋
- XP 부여: 정답 +10, 오답 +2
- 스트릭 업데이트: 오늘 날짜가 `streak.lastDate`와 다르면 갱신

### 계속 여부
"한 라운드 더 하시겠어요? (Y/N)"로 물어봅니다.

## 언어 규칙
- 모든 설명과 피드백은 **한국어**로 제공합니다.
- N4 이하 학습자에게는 반드시 **후리가나**를 병기합니다 (예: 食(た)べる).
