## 3. SRS 통합

### 3.1 핵심 변경: correct를 float로

```
// 정답 처리
concept.attempts += 1
concept.correct += weight  // 0.4~1.0 (소수점)

// 오답 처리
concept.attempts += 1
concept.correct += 0
```

### 3.2 nextReview 갱신 규칙

```
if (isCorrect) {
  if (gameWeight >= 0.7) {
    advanceToNextInterval();  // 1d->3d->7d->14d->30d->60d
  } else {
    keepCurrentInterval();    // 간격 유지, lastReview만 갱신
  }
} else {
  resetToOneDay();
}
```

### 3.3 승급 검증 (Promotion Gate)

SRS 간격 7d 이상 승격 조건: 가중치 0.7+ 게임 또는 표준 퀴즈에서 최소 1회 정답 필요.
- 낮은 가중치 게임(매칭카드 0.4)만으로는 간격 3d까지만 상승
- 근거: Karpicke(2008) — 쉬운 재인으로는 장기기억 고정 불충분

### 3.4 게이밍 방지

1. **동일 개념 동일 게임 24시간 쿨다운**
2. **게임 다양성 보너스**: 다른 게임 유형으로 학습 시 XP +5
3. **배지 확정 조건**: 최소 2가지 다른 평가 방식 필요
4. **마스터 개념 XP 반감**: 파랑 배지(마스터) 개념은 게임 XP 50%

## 4. 난이도 곡선 (게임당 4티어)

### 자동 배정 규칙

```
최근 3판 승률 기준:
- 90% 이상 → 다음 티어로 승격
- 70-89% → 유지
- 50-69% → 유지 (격려 메시지)
- 50% 미만 → 이전 티어로 강등
```

## 5. 스마트 추천 시스템

### 점수 공식

```
최종 점수 = 트리거 점수 + 신선도 점수 + 프로필 점수
```

### 5가지 트리거

| # | 트리거 | 발동 조건 | 최고 점수 |
|---|--------|-----------|-----------|
| 1 | 새 개념 학습 직후 | 신규 3개+ 학습 직후 | +90 |
| 2 | 복습 큐 과적 | nextReview ≤ 오늘 10개+ | +75 |
| 3 | 반복 피로 | 3세션 연속 같은 모드 | +85 |
| 4 | 약점 타겟팅 | 빨강 배지 3개+ 카테고리 | +95 |
| 5 | 데일리 마무리 | /jp:daily 마지막 | +100 |

### 신선도 감쇠

```
7일+ 미플레이: +20
3-6일 전: +10
1-2일 전: +0
오늘 1회: -30
오늘 2회+: -60
```

## 8. 안티패턴 방지

1. **유창성 환상** — 인코딩 게임 후 반드시 인출 게임/퀴즈 연결
2. **XP 파밍** — 마스터 개념 XP 50%, 동일 개념+게임 24h 쿨다운
3. **맥락 없는 반복** — 게임에 예문/문맥 포함, 문맥 기반 게임 적극 추천
4. **과잉 게이미피케이션** — "오늘 새로 알게 된 것" > XP 보고, 스트릭 freeze 1회/주
5. **난이도 회피** — 세션 아이템 30%는 시스템 강제 약점 배치
6. **벼락치기** — 10분 초과 XP 효율 50% 감소
7. **피드백 무시** — 오답 3초 강제 표시, 세션 종료 시 오답 복습
