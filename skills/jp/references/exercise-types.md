# 8가지 학습 모드 상세 규칙

각 학습 모드의 세션 흐름, 출제 규칙, 피드백 방식, 진도 기록을 정의합니다.

---

## 1. 어휘 퀴즈

### 세션 흐름
1. 레벨/주제 확인 (사용자 지정 또는 자동 선택)
2. 4문제 1라운드 출제 (quiz-rules.md 규칙 준수)
3. 문제별 즉시 피드백
4. 라운드 종료 → 결과 요약 → 계속 여부 확인

### 출제 규칙
- 현재 레벨의 어휘에서 출제
- ⬜(미측정) 개념 우선 → 🟥(약점) → 🟨(보통) 순
- 예문은 학습자 레벨에 맞는 문법으로 구성
- 한 라운드에 같은 카테고리(음식, 동사 등) 편중 금지

### 피드백 방식
- 정답: "⭕ 正解! [단어] — [뜻]" + 추가 예문 1개
- 오답: "❌ [정답 설명] + 혼동 포인트 해설" + 예문
- 해설은 한국어, 예문은 일본어 (후리가나 병기)

### 난이도 차이
- **N5-N4**: 기본 뜻 맞추기 중심, 히라가나 병기
- **N3**: 유의어 구분, 예문 빈칸 추가
- **N2-N1**: 문맥 속 적절한 어휘 선택, 관용 표현 포함

## 2. 문법 연습

### 세션 흐름
1. 문법 포인트 선택 (사용자 지정 또는 자동)
2. 패턴 설명 (구조 + 의미 + 예문 2-3개)
3. 연습 문제 4문제 (변환/선택/오류찾기)
4. 피드백 + 추가 연습 여부

### 출제 규칙
- 새 문법: 설명 → 쉬운 연습 → 응용 (단계적)
- 복습 문법: 바로 연습 문제 (설명 생략)
- 비슷한 문법끼리 비교 연습 포함 (예: ~たら/~ば/~と)

### 문제 유형
1. **변환**: 주어진 문장을 지정 형태로 변환 (예: 능동→수동)
2. **빈칸 완성**: 문장의 빈칸에 적절한 문법 표현 선택
3. **오류 찾기**: 문장 중 문법 오류 부분 지적 + 교정
4. **문장 만들기**: 주어진 단어와 문법으로 문장 구성 (주관식)

### 피드백 방식
- 정답: 올바른 이유 간략 설명
- 오답: 올바른 형태 + 왜 틀렸는지 + 비교 예문
- 주관식: 문법 정확성 위주 피드백, 모범 답안 제시

### 난이도 차이
- **N5-N4**: 단일 문법 포인트, 짧은 문장
- **N3**: 복합 문법, 비교 연습
- **N2-N1**: 뉘앙스 차이, 문어체/구어체 구분, 복합 표현

## 3. 한자 학습

### 세션 흐름
1. 한자 그룹 선택 (레벨별/부수별/주제별)
2. 플래시카드 형식 학습 (한자 → 읽기/뜻)
3. 확인 퀴즈 4문제
4. 결과 요약

### 출제 규칙
- 한 세션에 새 한자 5-8자 (과부하 방지)
- 부수/의미 그룹으로 묶어서 학습 (관련 한자끼리)
- 음독/훈독 모두 다루되, 일상적으로 쓰이는 읽기 우선

### 학습 요소 (한자당)
- 한자: 大
- 음독: タイ, ダイ
- 훈독: おお(きい)
- 뜻: 크다
- 부수: 大 (대)
- 획수: 3획
- 관련 단어: 大学(だいがく), 大きい(おおきい), 大人(おとな)
- 혼동 한자: 太(タイ/ふと-い), 犬(ケン/いぬ)

### 피드백 방식
- 읽기 오류: 올바른 읽기 + 혼동 원인 설명
- 뜻 오류: 올바른 뜻 + 관련 단어로 맥락 보강
- 비슷한 한자 비교표 제시

### 난이도 차이
- **N5**: 기본 한자 100자, 훈독 중심
- **N4**: 300자, 음독 비중 증가
- **N3-N1**: 복합어 읽기, 특수 읽기(熟字訓), 문맥 속 읽기

## 4. 독해

### 세션 흐름
1. 레벨에 맞는 지문 생성/제시
2. 지문 읽기 (모르는 단어는 질문 가능)
3. 이해 질문 2-3개
4. 핵심 어휘/문법 해설
5. (선택) 요약 작성 연습

### 지문 규칙
- 레벨별 길이: N5(3-5문장), N4(5-8문장), N3(10-15문장), N2-N1(1-2단락)
- 사용 문법/어휘는 학습자 레벨 ±1 범위
- 주제: 일상, 문화, 뉴스, 에세이 등 다양하게
- 자연스러운 일본어 (교과서 문체 탈피)

### 질문 유형
1. **사실 확인**: 지문에 명시된 정보 찾기
2. **추론**: 지문에서 유추할 수 있는 내용
3. **어휘 의미**: 밑줄 친 단어/표현의 의미 파악
4. **주제 파악**: 지문의 중심 내용 요약

### 피드백 방식
- 정답 여부 + 근거 (지문의 어디에서 알 수 있는지)
- 핵심 어휘 목록 (읽기 + 뜻)
- 주요 문법 포인트 해설
- 진도: 독해 주제별 이해도로 기록

### 난이도 차이
- **N5-N4**: 히라가나/가타카나 위주, 짧은 대화문이나 안내문
- **N3**: 신문 기사 스타일, 의견문
- **N2-N1**: 에세이, 논설문, 추상적 주제

## 5. 작문

### 세션 흐름
1. 주제/상황 제시 (또는 사용자가 자유 주제)
2. 사용자 작문 작성
3. 교정 피드백 (문법/어휘/자연스러움)
4. 모범 답안 제시 + 비교

### 주제 예시
- N5: "자기소개를 써보세요" / "오늘 한 일을 써보세요"
- N4: "좋아하는 계절과 이유" / "주말 계획"
- N3: "일본 문화에서 인상 깊었던 것" / "SNS의 장단점"
- N2-N1: "저출산 문제에 대한 의견" / "AI와 교육의 미래"

### 피드백 항목
1. **문법 정확성**: 문법 오류 지적 + 교정
2. **어휘 적절성**: 더 자연스러운 표현 제안
3. **자연스러움**: 일본어다운 표현인지 평가
4. **구성**: 논리적 흐름 (중급 이상)
- 각 항목별 간단한 점수 코멘트 (grading-rubrics.md 참조)

### 교정 형식
```
원문: 昨日は友達に会いて、映画を見ました。
교정: 昨日は友達に会って、映画を見ました。
설명: 「会う」는 5단 동사로 て형은 「会って」입니다 (촉음편).
```

### 난이도 차이
- **N5-N4**: 1-3문장, 기본 문형 사용
- **N3**: 단락 (5-10문장), 접속사 사용
- **N2-N1**: 에세이 (여러 단락), 문어체, 논리 구성

## 6. 번역

### 세션 흐름
1. 번역 방향 선택 (한→일 / 일→한)
2. 원문 제시
3. 사용자 번역
4. 피드백 (직역/의역 비교, 표현 선택)

### 출제 규칙
- 한→일: 한국어 문장 → 일본어로 번역 (작문 능력 평가)
- 일→한: 일본어 문장 → 한국어로 번역 (이해력 평가)
- 레벨에 맞는 문장 길이/복잡도
- 한 세션에 3-5문장

### 피드백 방식
- 사용자 번역 vs 모범 번역 비교
- 직역과 의역 둘 다 제시
- 문맥에 따른 표현 선택 해설 (예: "あげる" vs "くれる" vs "もらう")
- 어색한 번역투 지적 + 자연스러운 대안

### 진도 기록
- 번역 정확도를 문법/어휘/자연스러움으로 분리 평가
- 반복 오류 패턴 error_notes 기록

### 난이도 차이
- **N5-N4**: 단문, 일상적 내용
- **N3**: 복문, 접속 표현
- **N2-N1**: 관용 표현, 뉘앙스가 중요한 문장, 경어

## 7. 회화 시뮬레이션

### 세션 흐름
1. 상황 설정 (장소/관계/목적)
2. Claude가 상대방 역할, 사용자가 응답
3. 4-6턴 텍스트 대화
4. 대화 종료 → 사후 교정 + 대안 표현

### 상황 예시
- 식당에서 주문하기
- 편의점에서 물건 사기
- 회사에서 상사에게 보고하기
- 병원에서 증상 설명하기
- 부동산에서 방 구하기
- 전화로 예약하기

### 진행 규칙
- Claude는 자연스러운 일본어로 응답 (레벨에 맞게 조절)
- 사용자가 막히면 힌트 제공 (전체 답 아닌 키워드/구조 힌트)
- 경어/반말은 상황에 맞게 (상사↔부하, 점원↔손님 등)
- 대화 중에는 교정하지 않음 (흐름 유지)

### 사후 교정
대화 완료 후:
1. 사용자의 각 발화를 검토
2. 문법/어휘/경어 사용 교정
3. 더 자연스러운 대안 표현 제시
4. 잘한 점 강조

### 교정 형식
```
턴 3: 「すみません、これはいくらですか」
→ ✅ 적절한 표현입니다.
→ 💡 대안: 「すみません、お値段を伺ってもよろしいですか」(더 정중)

턴 4: 「ちょっと高いですね。安くしてください」
→ ⚠️ 「安くしてください」는 직접적이어서 실례가 될 수 있습니다.
→ 💡 대안: 「少しお値引きしていただけませんか」(부드러운 요청)
```

### 난이도 차이
- **N5-N4**: 단순 상황 (인사, 쇼핑), 정중체(ます형) 중심
- **N3**: 일상 상황, 반말/정중체 혼합
- **N2-N1**: 비즈니스 상황, 경어/겸양어/존경어 구분

## 8. 문화/관용표현

### 세션 흐름
1. 주제 선택 (사용자 지정 또는 자동)
2. 문화 배경 설명 (한국어)
3. 관련 표현/관용구 학습 (일본어 + 해설)
4. 확인 퀴즈 2-3문제

### 주제 카테고리
- **인사/예절**: お辞儀, 名刺交換, 靴を脱ぐ
- **비즈니스**: 敬語の使い分け, メール作法, 報連相
- **일상**: 割り勘, お土産文化, 空気を読む
- **계절/행사**: お正月, お花見, お盆
- **관용구/속담**: 猫の手も借りたい, 花より団子, 七転び八起き

### 학습 요소 (관용구당)
- 표현: 猫の手も借りたい
- 읽기: ねこのてもかりたい
- 직역: 고양이의 손이라도 빌리고 싶다
- 의미: 매우 바빠서 누구라도 도움이 필요하다
- 사용 예: 「年末は猫の手も借りたいほど忙しい」
- 한국어 대응: "손이 열 개라도 모자란다"

### 피드백 방식
- 문화 맥락 이해 확인
- 상황별 적절한 표현 선택 퀴즈
- 한국 문화와의 비교/대조로 이해 촉진

### 난이도 차이
- **N5-N4**: 기본 인사, 일상 매너
- **N3**: 사회생활 관련 문화, 기본 관용구
- **N2-N1**: 비즈니스 문화, 고급 관용구/속담, 뉘앙스
